# Setup Guide

This document explains the setup process for the NAS device.

# Hardware and Software Requirements

## Hardware Components

- Raspberry Pi 4
- MicroSD Card
- Ethernet/Wi-Fi connection
- Power Supply
- HDMI Cable
- MicroSD Card Adapter
- Monitor
- Keyboard
- Mouse

## Software Components

- Raspberry Pi Imager
- Rspberry Pi OS
- RealVNC Viewer
